# 논문 검색 및 선별

사용자가 학술 논문을 검색하고 선별하는 작업을 요청했습니다.

## 워크플로우

다음 단계를 순서대로 수행하세요:

### 1단계: 요구사항 수집

사용자에게 다음 정보를 물어보세요 (이미 제공된 정보는 건너뛰세요):

1. **연구 주제**: 어떤 주제의 논문을 찾고 있나요?
2. **검색 목적**: 논문을 찾는 구체적인 목적이 무엇인가요?
   - 최신 연구 동향 파악
   - 특정 방법론/기술 탐색
   - 선행연구 조사
   - 리뷰 논문 작성
   - 기타
3. **논문 수**: 몇 편의 논문을 검토하길 원하나요? (기본: 30편)
4. **연도 범위**: 특정 연도 이후의 논문만 원하나요? (선택사항)
5. **추가/제외 키워드**: 반드시 포함하거나 제외할 키워드가 있나요? (선택사항)

### 2단계: 검색 실행

정보를 수집한 후 다음 명령을 실행하세요:

```bash
python search_papers.py --topic "[주제]" --count [수] [옵션들]
```

사용 가능한 옵션:
- `--year-from YYYY`: 특정 연도 이후 논문만
- `--year-to YYYY`: 특정 연도 이전 논문만
- `--additional "키워드1,키워드2"`: 추가 키워드
- `--exclude "키워드1,키워드2"`: 제외할 키워드

### 3단계: 결과 분석 및 선별

검색 결과 출력을 분석하여 다음 기준으로 논문을 평가하세요:

**관련성 (높음/중간/낮음)**
- 제목과 초록이 사용자의 검색 목적과 얼마나 관련이 있는가?

**품질 지표**
- 인용 횟수 (분야와 출판연도 대비)
- 게재 저널/학회의 명성

**목적 적합성**
- 사용자가 명시한 목적에 얼마나 부합하는가?

### 4단계: 결과 보고

다음 형식으로 결과를 정리하세요:

```markdown
## 📚 논문 검색 결과

### 검색 조건
- 주제: [입력한 주제]
- 목적: [검색 목적]
- 검색 쿼리: `[생성된 쿼리]`
- 결과: 총 N편 검색, M편 추천

---

### ⭐ 강력 추천 (상위 5편)

#### 1. [논문 제목]
- 저자: [저자명]
- 저널: [저널명] ([연도])
- 인용: [N]회
- DOI: https://doi.org/[DOI]
- **추천 이유**: [2-3문장으로 왜 이 논문이 사용자 목적에 적합한지]
- **핵심 내용**: [초록 기반 2-3문장 요약]

---

### 📋 추가 검토 권장 (다음 5-10편)

[간략 형식으로 나열]

---

### 💡 추가 검색 제안

[더 나은 결과를 위한 키워드나 필터 제안]
```

## 참고사항

- 검색 결과는 `data/papers/` 디렉토리에 자동 저장됩니다
- 저장된 결과는 `--load` 옵션으로 다시 불러올 수 있습니다
- `SCOPUS_API_KEY` 환경변수가 설정되어 있어야 합니다
